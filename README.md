# 👁 SUHO
> 당신의 대화를 지켜주는 AI

---

## 왜 만들었나

대화가 많아지면 놓치는 게 많다. 말이 빠르면 따라가기 힘들고, 뭐라고 대답해야 할지 모르겠고, 지시사항은 기억이 안 나고, 모르는 부분을 되물어야 하는데 그것도 모르겠다.

보청기도 아니고, 번역기도 아니다. **대화 중에 조용히 옆에서 도와주는 AI.**

신입 직장인, 사회적 소통이 어려운 사람, 정보 처리가 힘든 사람 모두를 위한 실시간 대화 보조 시스템.

---

## 이름의 유래

**SUHO(수호)** — 守護. 지키다, 보호하다.

수호령(守護靈)에서 따왔다. 나만 볼 수 있는 존재가 옆에서 조용히 도와주는 느낌. 이세계 소설의 시스템창처럼.

---

## 타겟

- 대화 맥락 파악이 어려운 **신입 직장인**
- ASD, ADHD 등 **신경다양인**
- 빠른 말을 따라가기 힘든 **일반인**

---

## 핵심 기능

| 기능 | 설명 |
|---|---|
| 실시간 자막 | 상대 말을 텍스트로 변환 |
| 요약 | 긴 말 → 핵심 3줄 |
| 응답 제안 | 어떻게 대답할지 바로 제안 |
| 맥락 파악 | 지금 상대가 원하는 게 뭔지 |
| 장난/빈정거림 감지 | 말의 표면이 아닌 의도 파악 |
| 할일 자동 저장 | 지시사항 → 자동으로 할일 목록 |
| 대화 기억 | 이전 대화 맥락 오늘 대화에 반영 |

---

## 하드웨어

- **지금**: 갤럭시 + 크롬 브라우저 (마이크)
- **목표**: Rokid Glasses AR 안경 + 갤럭시

안경이어야 하는 이유: 카메라로 상대 표정·상황 실시간 감지 가능. 말 + 표정 동시 분석으로 정확도 향상.

## 차별점

1. **실시간** — 대화 중에 작동
2. **자동 상황 감지** — 모드 전환 없음
3. **안경** — 표정·상황까지 감지 (예정)
4. **구글 연동** — 개인정보 구글이 책임
5. **AI 선택** — Claude, ChatGPT, Gemini 골라 쓰기
6. **할일 자동 생성** — 지시사항 → 바로 목록화

## 로드맵

**Phase 1 (현재)** — 폰 프로토타입
- 마이크 + Gemini AI 실시간 분석
- 웹앱으로 즉시 사용 가능

**Phase 2** — 앱 완성
- 구글 로그인 연동
- 대화 기록 자동 저장
- 할일 → 구글 캘린더 연동

**Phase 3** — 안경 연동
- Rokid Glasses SDK 연동
- 카메라 표정 분석
- 렌즈 디스플레이 최적화

**Phase 4** — 상용화
- 구독 모델 (월 1~2만원)
- B2B (신입 온보딩 툴로 기업 판매)
